# Mocha / chai test
* [Mocha](https://mochajs.org/)
* [chai](https://www.chaijs.com/)

## Install
```
npm install mocha
```
```
npm install mocha -g
```
```
npm install chai
```

## Run test manualy
* If test files are put into test dir then run:
```
$misc_js/mocha-chai-unit-testing mocha
```
* Run file seperatly
```
mocha test.js 
```
